                          UNITY WEBGL CONFIGURATION

-A se deschide browserul Mozile Firefox 
-Se introduce comanda about:config in partea de URL a browserului
-In pagina se va deschide o bara in care se introduce urmatoare comanda:
	security.fileuri.strict_origin_policy
-Dupa ce apare aceasta comanda se seteaza pe FALSE.
-Dupa ce setarea a fost facuta se deschide fisierul index.html
